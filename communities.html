Roblox Communities

  • ORC Reddit

    The ORC Reddit is probably one of the most popular discussion forums in the community right now with a total of 5k+ members.

    Pros:

    • Active members

    Cons:

    • Moderators arent doing their job properly (alot of miss information is currently being spread there and there is barely any action being taken)

  • Rchive

    Rchive is a community that is dedicated to preserving old Roblox media. This is a great recourse for old Roblox.

    Pros:

    • Great recourse/source
    • Diverse

    Cons:

    • I haven't seen any cons of this community, let me know if there are any.

  • Legacy Host

    Legacy Host is a small Discord server where members host games for other members using various launchers, such as Novetus or Sodikm.

    Pros:

    • Active community

    Cons:

    • Small community

  • ORC Guide

    ORC Guide is a Discord server with various tutorials on how to create your own revival, along with many different clients, RCC, and more.

    Pros:

    • Active community
    • A lot of useful files and tutorials

    Cons:

    • Small community
