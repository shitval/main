Roblox Revival List

  • Pekora

    Pekora is a 2017-2021 revival that allows you to play various old clients. Pekora is currently available to play on Windows, iOS and Android.

    Pros:

    • Big community
    • Active community

  • Solario

    soon...

    Pros:

    • Open-Source bootstrapper
    • Mobile Support
    • Active Community

    Cons:

    • Only the 2021 client is available
    • Private testing

  • Hexagon

    Hexagon is a 2014 revival with custom source code, that revival is currently available for Windows and iOS.

    Pros:

    • The only revival at the moment that uses a client older than 2016.
    • Active community

    Cons:

    • I haven't noticed any cons of this revival so far. If there are any, feel free to send a complaint.

  • Boomblox

    Boomblox is a 2008 revival that uses the 2008L client, with its own source code and pages styled like the old Roblox from those years.

    Pros:

    • The only revival that uses a client older than 2012.

    Cons:

    • Small community
    • Private testing

  • Uriven

    Uriven is a Syntax based revival that is currently available only for Windows. This revival is currently a 2021 revival.

    Pros:

    • Active development

    Cons:

    • Only the 2021 client is available

  • ECS:R

    Economy Simulator: Reborn is a heavily modified project based on the original source code of the Economy Simulator, which was made available by the former admin of ECS. Currently, only the 2016 client is available. ECS:R is currenly available for Windows, Android and iOS.

    Pros:

    • Big community
    • Active community

  • idk16

    IDK16 is an old Roblox revival, trying to revive the era of late 2016. idk16 is trying to use original Roblox software for playing from this period of time, ensuring there is the best accuracy with the experience. idk16 currently support Windows, Android, Linux with Wine and iOS.

    Pros:

    • Supports android, iOS, linux (wine)
    • Owner does testing events sometimes

    Cons:

    • In testing
    • Community isn't that active
